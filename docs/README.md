## **功能**

- 支持Github/码云平台的片段管理, 拥有无限公共/私有片段, 云同步, 在线编辑, 分享他人
- 可对片段分组
- 即时运行指令
- 运行动态脚本片段
- 自定义日期、模拟键盘宏、生成富文本模板
- 片段代码高亮、格式化、替换等
- 生成代码图片
- 片段全文搜索
- 代理服务
- 跨平台
  ...
