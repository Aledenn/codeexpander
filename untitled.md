# 环境安装

## 首次安装

For windows:

> 只需要配置相关的 Python 环境变量就完成安装.

For mac:

初次启动软件时候，软件会提示如下：

![](.gitbook/assets/yin-si-an-zhuang.png)

打开系统偏好设置，勾选 \`Oncework\`

![](.gitbook/assets/yin-si-an-zhuang-2.png)

{% hint style="info" %}
温馨提示: 设置后一定要重启软件，否则无法进行正常软件运行.
{% endhint %}

## 编译功能环境支持

```text
$ node -v
$ sh
$ python -v
```

{% hint style="info" %}
若缺少其中的某些命令，请自行下载安装。
{% endhint %}
